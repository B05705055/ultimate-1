GENERAL INFORMATION

CDTDummyParser is a part of ULTIMATE project.
Dummy parser is used to pass the already parsed AST through Ultimate!
This is needed because we have to start Ultimate with a Source Plugin, which
should generate some Output. So here we only use the a priori set AST and give it
back.
