Automata Library

The Ultimate Automata library provides implementations of the following automaton models, together with many operations (listed further below).

  • Nested word automata, aka visibly pushdown automata (we consider the weakly-hierarchical variant)
  • Finite automata (a special case of nested word automata)
  • Büchi nested word automata (we consider the weakly-hierarchical variant)
  • Büchi automata (a special case of Büchi nested word automata)
  • Petri net automata (i.e., Petri nets used as language acceptors)
  • Alternating finite automata
  • Tree automata

The Automata library is maintained by Matthias Heizmann and Christian Schilling.

Web Interface

In addition to the Java interface the Automata library also offers a text interface called Automata Script.
Ultimate provides an interpreter for Automata Script that is also available via a web interface.

News

Minimization based on partial Max-SAT for nested word automata available (2017-02-09)

The new operation minimizeNwaPmaxSat reduces the size of a nondeterministic nested word automaton. It uses a domain-specific partial Max-SAT solver in the background. An example automata script file will soon be available in our web interface.

The underlying theory is presented here: extended version.

Optimized complementation for semi-deterministic Büchi automata available (2015-10-18)

The operation buchiComplementNCSB implements the algorithm of the Complementing Semi-deterministic Büchi Automata paper which was published at TACAS 2016. (Additional material: preprint, files to reproduce evaluation, slides) The operation isSemiDeterministic can be used to check if an automaton is semi-deterministic. An example automata script file is available in our web interface.

Computation of loop complexity available (2015-09-15)

The new operation loopComplexity computes the loop complexity of finite automata. An example automata script file is available in our web interface.
