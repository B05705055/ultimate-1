Automata Script Interpreter

The automata script interpreter is a user interface of the Ultimate Automata Library

.

This library provides implementations for nested word automata, Büchi nested word automata, Petri net automata, and finite alternating automata.

News

Optimized complementation for semi-deterministic Büchi automata available

(2015-10-18) The new operation 

buchiComplementNCSB
constructs the complementation of a semi-deterministic Büchi automaton. The operation 
isSemiDeterministic
can be used to check if an automaton is semi-deterministic. An example automata script file is available in our web interface.

Computation of loop complexity available

(2015-09-15) The new operation 

loopComplexity
computes the loop complexity of finite automata. An example automata script file is available in our web interface.

The Automata Script Interpreter is available via a web interface.
