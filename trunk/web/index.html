Ultimate

Ultimate is a program analysis framework. Ultimate consists of several plugins that perform steps of an program analysis, e.g., parsing source code, transforming programs from one representation to another, or analyze programs. Toolchains of these plugins can perform complex tasks, e.g., verify that a C program fulfills a given specification.

Web Interface

Several toolchains are available via the Ultimate web interface.

In the web interface the toolchains are grouped by the task that they accomplish. The currently the following tasks are available.

  • Verification of C programs
  • Verification of Boogie programs
  • Termination analysis for C programs (currently only Lasso Programs supported)
  • Termination analysis for Boogie programs (currently only Lasso Programs supported)

Toolchains

Some of the toolchains currently available are:

Ultimate Automizer

The Ultimate Automizer toolchain is an implementation of trace abstraction and can verify C programs and Boogie programs.
You can try out Ultimate Automizer in the web interface.

Ultimate LassoRanker

The Ultimate LassoRanker toolchain is able to synthesize ranking functions of lasso programs.
You can try out Ultimate LassoRanker in the web interface.

The core of Ultimate and many plugins are subject to the LGPLv3 license with a linking exception to Eclipse RCP and Eclipse CDT. We are planing to release the source code of Ultimate on this website, meanwhile the source code is available on request.

Dependencies

Many plugins in Ultimate require an SMT-LIB compatible theorem prover. We use SMTInterpol.

Ultimate uses Eclipse RCP. Plugins that handle C code depend on Eclipse CDT.
